# Job Listing Web Application

This is a full-stack job listing web application built using **Next.js**, **TypeScript**, **Tailwind CSS**, **Express.js**, and **MongoDB Atlas**.

---

## 🌐 Live Demo Links

- 🔗 **Frontend**:  [https://your-frontend-url.vercel.app](mployee-jobs.vercel.app)
- 🔗 **Backend (API)**:  [https://your-backend-url.vercel.app/api/jobs](mployee-jobs-1mzu.vercel.app)

> ⚠️ Note: The backend is deployed and connected to MongoDB, but due to an issue, the job list is currently not appearing on the frontend. All other functionality is implemented.

---

## 📦 Tech Stack

- **Frontend**: Next.js (TypeScript), Tailwind CSS
- **Backend**: Express.js with TypeScript
- **Database**: MongoDB Atlas
- **Deployment**: Vercel (Frontend & Backend)

## 🚀 Getting Started Locally

### 1. Clone the repository

```bash
git clone https://github.com/LoneExpert/Mployee-Jobs.git
cd Mployee-Jobs
```
### 2. Setup Backend

```bash
cd backend
npm install
npm run build
npm run dev
```

### 3. Environment Variables

```
MONGODB_URI=your_mongodb_uri
PORT=5000
```

### 4. Setup Frontend

```bash
cd frontend
npm install
npm run dev

const BASE_URL = "https://your-backend-url.vercel.app"
```

### 6. Folder Structure
```bash
Mployee-Jobs/
├── backend/         # Express.js backend
├── frontend/        # Next.js frontend
```

### 5. Features
  => List of job postings (fetched from MongoDB)<br/>
  => Click to view job details<br/>
